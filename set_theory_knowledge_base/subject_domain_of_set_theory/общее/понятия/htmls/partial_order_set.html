Частично упорядоченным множеством P называется система P{..., x, y, ...} элементов с отношением порядка x ≥ y (читается: "x включает y") для некоторых пар элементов и равенством x = y, в которой выполнены следующие аксиомы:

  • x ≥ x для ∀x из P;
  • Если x ≥ y и y ≥ z, то x≥z;
  • Если x ≥ y и y ≥ x, то x = y.
