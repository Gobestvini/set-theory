Топологиечское пространство - множество с дополнительной структурой определённого типа.
Пусть дано множество Χ. Система Τ его подмножеств называется топологией на Χ, если выполняются следующие условия:

  1. Объединение произвольного семейства множеств, принадлежащих Τ принадлежит Τ, то есть если Uα ∈ Τ ∀a ∈ A, то ∪α∈A Uα ∈ Τ.
  2. Пересечение конечного семейства множеств, принадлежащих Τ принадлежит Τ, то есть если Ui ∈ Τ i = 1,..., n, то ∩ni=1 Ui ∈ Τ.
  3. Χ, ∅ ∈ Τ.
Пара ( Χ, Τ) называется топологическим пространством. Множества, принадлежащие Τ, называются открытыми множествами.
