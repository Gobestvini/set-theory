Линейные рекурентные уравнения - это рекурентные уравнения вида
F(n+k) = ak-1F(n+k-1) + ak-2F(n+k-2) + ... + a1F(n+1) + a0F(n) + ƒ(n), где a0, a1, ..., ak-1 - некоторые числа(постоянные коэффициенты), а ƒ(n) - некоторая функция от n.
Такие уравнения называются однородными потому, что элементы последовательности F(n), ..., F(n+k-2), F(n+k-1), F(n+k) связаны линейной зависимостью.
Если при это функция ƒ(n) = 0, то уравнение такого вида называется однородным рекуррентным уравнение. В противном случае уравнение называется неоднородным.
