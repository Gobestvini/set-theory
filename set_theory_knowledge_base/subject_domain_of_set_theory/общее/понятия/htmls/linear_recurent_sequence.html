Линейная рекуррентная последовательность - это всякая числовая последовательность x0, x1, ..., задаваемая линейным рекуррентным соотношением:
xn = ∑di=1 aixn-i для всех n ≥ d
с заданными начальными числами x0, ..., xd-1, где d - фиксированное натуральное число, a1, ..., ad - заданные числовые коэффициенты, ad ≠ 0. При этом число d называется порядком последовательности.
Линейные рекуррентные последовательности иногда называют также возвратными последовательностями.
