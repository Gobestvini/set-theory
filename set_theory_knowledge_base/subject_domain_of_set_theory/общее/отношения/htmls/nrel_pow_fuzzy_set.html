Пусть дано нечёткое множества A с функцией принадлежности μA(x) для ∀ x ∈ X, где X - универсум. Возведением в степень n множества А называется множество An, которому принадлежат элементы из множества А со степенями принадлежности, равными степеням принадлежности элементов множества А, возведенным в n-ую степень, т.е. An = {μAn x | μAn = μAn(x)}.
