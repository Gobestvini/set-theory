Концетрация нечеткого множества - частный случай возведения в степень нечеткого множества. Пусть дано нечёткое множество A с функцией принадлежности для каждого x ∈ X μA(x), где X - универсум. Концентрацией множества А называется множество A2, которому принадлежат элементы из множества А со степенями принадлежности, равными степеням принадлежности элементов множества А, возведенным в квадрат, т.е. A2 = {μA2x | μA2 = μA2(x)}.
