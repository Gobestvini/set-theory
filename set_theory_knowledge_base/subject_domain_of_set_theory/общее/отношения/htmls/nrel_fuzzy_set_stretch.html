Растяжение нечеткого множества - частный случай возведения в степень нечеткого множества. Пусть дано нечёткое множество A с функцией принадлежности для каждого x ∈ X μA(x), где X - универсум. Растяжением множества А называется множество A0.5, которому принадлежат элементы из множества А со степенями принадлежности, равными степеням принадлежности элементов множества А, возведенным в степень 0.5, т.е. A0.5 = {μA0.5x | μA0.5 = μA0.5(x)}.
